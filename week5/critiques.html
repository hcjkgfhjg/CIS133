Web Design Feedbacks

This is a really simple website and really easy to navigate. Three things I like about this website are:

  1. Homepage downloads really fast, almost instantly
  2. Good use of basic design principles
  3. Navigation is easy, even for new users

However, despite the website being simple, there are a lot of things that could have been done better such as:

  1. The header is not consistent for all the pages
  2. Only one picture shows the resort, the rest looks like just random pictures on the internet
  3. Doesn't look appealing to target audience, it looks too simple and not a lot of work was put into it

With that being said, there are a couple of things I can think of that might make the website looks more appealing.

  1. Provide more pictures about the resort so your guests know what they're getting.
  2. Add animations to make navigating the website more fun and more interacting.
  3. Making sure everything is aligned on every webpages
  4. Add a "about us" page to have a better introduction instead of having the mission statement in the homepage
